「Webコンテンツ作成」の成果

本テーマについて

  • 目的

    • 先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得すること。

  • 到達目標
    1. 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できること。
    2. コミュニケーション能力の重要性を理解し、グループワークを円滑に実施できること。
    3. HTML等を用いたWebページ作成ができる。なお、本授業は、リーダーシップ能力、課題に対する解決能力育成に寄与する科目です。

      プロジェクト課題について

      今回のプロジェクトでは、グルメ情報を提供するWebページの作成に取り組みました。HTMLとCSSを使って見やすく操作しやすい画面を作り、JavaScriptで動きをつけています。特にAxiosというライブラリを使い、外部のAPIから飲食店の情報を非同期で取得できるようにしました。ユーザーが選んだジャンルに合わせて、関連するお店の情報を画面にリアルタイムで表示する機能を実装しています。これにより、利用者は簡単に好みのジャンルの飲食店を探せるようになり、快適に使えるサイトに仕上げることができました。

      各授業回の学習事項

      第1回 WWW と HTML

      今回の講義ではHTMLの基本的な知識、文章の構成とHTMLを使ってWebページを作成する方法を学びました。また、画像の添付方法、HTML要素の使い方、要素でハイパーリンクを設置する方法やWebページのデバッグの仕方を学びました。

      演習の成果:演習1-4: HTMLのさまざまな要素

      第2回 CSS による Web ページの装飾

      今回の講義ではCSSの基本的な知識、CSSを使ってWebページを装飾する方法を学びました。CSSのセレクタ、プロパティ、値の使い方、外部スタイルシートの作成方法、インラインスタイルと内部スタイルの違いについて学びました。

      演習の成果:演習2-3: ウェブブラウザ説明ページの装飾

      第3回 JavaScript の基本

      今回の講義ではJavaScriptの基本的な知識、JavaScriptを使ってWebページに動きを加える方法を学びました。コンソールを使ってエラーを探し、修正する方法を学びました。また、JavaとJavaScriptの二つの言語の構文の違いとJavaScriptのデバッグの仕方を学びました。

      演習の成果:オブジェクトの参照

      第4回 DOM と JavaScript

      今回の講義では、HTMLのinput要素の種類と属性、JavaScriptを使ったDOM操作の基本を学びました。querySelectorによる要素検索、textContentを使ったテキスト操作、addEventListenerを用いたイベント処理など、動的なWebページ作成の基礎を習得。特に、ボタンクリック時にフォームの入力値を取得して表示する実践的なインタラクションを作成することで、イベント駆動型プログラミングの重要な概念を理解しました。

      演習の成果:演習4-10: 足し算ページ

      第5回 イベント処理

      今回の講義では、JavaScriptを使ったDOM操作の基本を学びました。'createElement'で新しい要素を作成し、'insertAdjacentElement'を使ってリストに追加する方法、'setAttribute'で属性を変更する方法、'remove'で要素を削除する方法を実際に試しました。また、複数の要素を組み合わせて新しい構造を作り、ページに追加する練習も行いました。

      演習の成果:演習5-7: DOMの演習その2

      工夫点，苦心点，強調点，感想

      工夫点: 各回の学習内容を実際のWebページ作成に活かすため、実践的な演習を多く取り入れました。特に、JavaScriptを使った動的な要素の操作やイベント処理に重点を置きました。

      苦心点: JavaScriptの非同期処理やDOM操作に関する理解が難しいと感じる学生が多かったため、具体的な例を用いて説明を行い、実際に手を動かすことで理解を深めるよう努めました。

      強調点: Webページ作成においては、HTML、CSS、JavaScriptの三つの技術が密接に連携していることを強調しました。特に、DOM操作を通じて、JavaScriptがどのようにHTML要素と相互作用するかを理解することが重要です。

      感想: 学生たちが実際に手を動かしてWebページを作成することで、理論だけでなく実践的なスキルも身につけられたと感じています。また、グループワークを通じてコミュニケーション能力の向上も見られ、授業全体として充実した内容になったと思います。
