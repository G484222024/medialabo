本テーマについて

  • 目的

    • 先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得すること

  • 到達目標
    1. 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できること
    2. コミュニケーション能力の重要性を理解し、グループワークを円滑に実施できること
    3. HTML等を用いたWebページ作成ができる。なお、本授業は、リーダーシップ能力、課題に対する解決能力育成に寄与する科目です。

      各授業回の学習事項

      第1回 WWW と HTML

      今回の講義ではHTMLの基本的な知識、文章の構成とHTMLを使ってWebページを作成する方法を学びました。また、画像の添付方法、HTML要素の使い方、要素でハイパーリンクを設置する方法やデバッグの仕方を学びました。

      演習の成果:演習1-4: HTMLのさまざまな要素

      第2回 CSS による Web ページの装飾

      今回の講義ではCSSの基本的な知識、CSSを使ってWebページを装飾する方法を学びました。CSSのセレクタ、プロパティ、値の使い方、外部スタイルシートの作成方法、インラインスタイルと内部スタイルの違いについて学びました。

      演習の成果:演習2-3: ウェブブラウザ説明ページの装飾
